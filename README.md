# Titanfall-2-Custom-Skins-Archive
Version control for custom weapon skins made by me to be used in Titanfall 2. The purpose of this repository is to allow me to keep track of my own work while also making skins available to anyone who wants to use them.

To use any of these skins in your game, dowload the Titanfall 2 skin tool here: https://github.com/zxcPandora/Titanfall2-SkinTool/releases

Once the tool is installed and configured, navigate to the .zip of a skin you have downloaded by pressing the three dots at the top right then selecting "Import Skin".
