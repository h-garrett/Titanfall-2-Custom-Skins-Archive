# Titanfall-2-Custom-Skins-Archive
Version control for custom weapon skins made by me to be used in Titanfall 2

To use any of these skins in your game, dowload the Titanfall 2 skin tool [here]([url](https://github.com/zxcPandora/Titanfall2-SkinTool/releases))

Once the tool is installed and configured, navigate to the .zip of a skin you have downloaded by pressing the three dots at the top right then selecting "Import Skin".
